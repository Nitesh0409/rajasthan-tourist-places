#  Tourist Places of Rajasthan  
### Interactive Map Web Project â€” Digital Society (LAL3080)

This project is an interactive web-based map that showcases major tourist locations across Rajasthan.  
Users can explore the map by clicking on hotspots to view images, short descriptions, and detailed information about forts, palaces, temples, museums, lakes, markets, wildlife locations, and many other attractions.

This project was created as part of the **Digital Society (LAL3080)** course and demonstrates how digital information can be visually represented and made interactive using modern web technologies.
